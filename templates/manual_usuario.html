📖 Introdução

Bem-vindo ao Sistema NOC/CGR da Vision Online! Este é um sistema completo de gerenciamento de redes que integra múltiplas tecnologias para facilitar a administração de sua infraestrutura.

O que este sistema faz?

  • 🔧 Gerencia equipamentos de rede - Huawei, Cisco, Mikrotik e outros
  • 🔍 Monitora via SNMP - Integração com LibreNMS
  • 💾 Backup automático - Configurações salvas diariamente
  • 🤖 Assistente de IA - Claude AI integrado para ajudar em tudo
  • 🌐 Acesso remoto - SSH/Telnet via web
  • 📊 Relatórios e análises - Insights inteligentes
✅ Versão Atual: Sistema em produção com IA Claude integrada desde fevereiro de 2026.

🖥️ Gerenciamento de Dispositivos

Adicionar Novo Equipamento

  1. Acesse o menu "Dispositivos"
  2. Clique em "Adicionar Dispositivo"
  3. Preencha os campos obrigatórios:
    • Nome - Hostname do equipamento
    • IP - Endereço IP de gerência
    • Vendor - Fabricante (Huawei, Cisco, etc.)
    • Usuário/Senha - Credenciais SSH/Telnet
    • Porta - Padrão: 22 (SSH) ou 23 (Telnet)
  4. Configure SNMP (opcional mas recomendado)
  5. Salve
💡 Dica: Use a IA! Diga "Adicione o equipamento 192.168.1.10 com usuário admin" e ela te guiará no processo.

  • Histórico de comandos

Consultas SNMP

Execute queries SNMP diretamente na interface web.

💾 Backups Automáticos

Como Funciona

O sistema conecta via SSH/Telnet em cada equipamento e salva a configuração completa automaticamente.

Configuração

  • Frequência: Diária (padrão às 03:00)
  • Retenção: 30 dias
  • Formato: Texto puro (.cfg)
  • Local: /opt/lorcgr/backups/
✅ Dica com IA: Pergunte "Quando foi o último backup do CORE-01?" ou "Restaure a configuração de ontem do R2".

📊 Monitoramento com LibreNMS

O sistema integra com LibreNMS para coleta de métricas SNMP.

Métricas Coletadas

  • 📈 Tráfego de interfaces
  • 🔥 CPU e Memória
  • 🌡️ Temperatura
  • 💡 Status de interfaces (UP/DOWN)
  • ⚠️ Erros e descartes

❓ Perguntas Frequentes (FAQ)

A IA pode executar comandos sem eu confirmar?

Não. Para comandos que alteram configuração, a IA sempre pede confirmação explícita antes de executar.

A IA aprende com minhas conversas?

Não. Cada conversa é independente. A IA não retém informações entre sessões por questões de privacidade e segurança.

Posso usar a IA em português?

Sim! A IA entende perfeitamente português brasileiro e responde no mesmo idioma.

O que fazer se a IA não conseguir acessar um equipamento?

Verifique:

  • ✅ Credenciais (usuário/senha) estão corretas
  • ✅ IP do equipamento está acessível
  • ✅ Porta SSH/Telnet correta (22/23)
  • ✅ Firewall não está bloqueando
Depois pergunte à IA: "Por que não consegue acessar o R1?" e ela ajudará no diagnóstico.

Posso solicitar relatórios personalizados?

Sim! Exemplo: "Gere um relatório de todos os equipamentos Huawei com CPU acima de 70%"

A IA substitui o conhecimento técnico?

Não. A IA é uma ferramenta poderosa de auxílio, mas o conhecimento e validação final são sempre do engenheiro de redes.
