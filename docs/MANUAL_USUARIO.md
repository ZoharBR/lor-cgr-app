# Manual do Usuário - LOR CGR

## Terminal SSH

### Conectar
1. Menu -> Terminal
2. Clique em + Nova
3. Selecione o dispositivo

### Teclas
- TAB: Autocompletar
- BACKSPACE: Apagar atrás
- DELETE: Apagar frente
- Ctrl+C: Cancelar

### Configurações
- Clique na engrenagem
- Ajuste fonte e cor
- Clique em Salvar

## IA Assistente

### Usar
1. Menu -> IA Assistente
2. Digite sua pergunta
3. Aguarde resposta

### Exemplos
- Qual comando ver interfaces no Huawei?
- Como diagnosticar cliente PPPoE?
- Configurar VLAN no Mikrotik

## Backups

### Backup Manual
1. Menu -> Backups
2. Selecione dispositivo
3. Clique Executar

### Restaurar
1. Localize o backup
2. Clique em Restaurar
